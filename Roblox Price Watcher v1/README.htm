Roblox Price Watcher v1.1.0

Date: March 6, 2025

Roblox Price Watcher v1.1.0 is a desktop application designed to monitor the prices of Roblox limited items. Built with C++, DirectX, and ImGui, it provides a user-friendly interface to track asset prices, set thresholds, and receive Discord notifications when prices drop below user-defined limits. This version includes core functionality with more features planned for future releases.

Current Features:

  • Track Roblox limited items by Asset ID with real-time price monitoring.
  • Set custom price thresholds for each asset to trigger notifications.
  • Receive Discord webhook alerts when an asset's price falls below its threshold, formatted for clear readability.
  • Visualize price history with an interactive graph showing price changes over time.
  • Persistent storage of assets, cookies, and price history in local cache files.
  • Modern dark-themed UI with docking support and customizable check intervals.
  • Thread-safe price checking with concurrent API requests for efficiency.

Future Plans:

This is just the beginning! More features are to be added, including an entire proxy system planned for a future release. The proxy system will allow users to access Roblox APIs without hitting rate limits, enabling seamless and uninterrupted price monitoring for a larger number of assets. Stay tuned for updates!
